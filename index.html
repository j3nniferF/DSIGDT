    About

    DSIGDT helps you organize tasks, focus with a timer, and celebrate progress.

    Tasks

    • Add tasks per tab
    • Edit task text by double-clicking a task
    • Mark complete, uncheck, or delete tasks anytime
    • Completed tasks move into the done list by tab

    Tabs

    • Start with 4 default tabs for separate task groups
    • Rename tabs by clicking the active tab name
    • Add new tabs or delete tabs you do not need
    • Use All Tasks view to see everything together

    Timer

    • Choose an incomplete task before launching timer
    • Set HRS / MIN / SEC with dial-style wheels
    • Floating timer has Start, Pause, and Reset controls
    • Timer completion marks that selected task as done

    Rewards + Progress

    • Confetti + sound celebrate completed work sessions
    • Reward popup appears after full-tab or timer completion
    • Statistics track completed tasks and focus sessions
    • Reset All clears tasks, tabs, completed lists, and stats
    • Data saves automatically in your browser storage
